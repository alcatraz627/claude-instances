# V2 platform specs (abandoned, kept for reference)

`widget-system.md` and `dashboard-kit.md` describe a generic, plugin-based widget
**platform** — a separate, abandoned direction, NOT this claude-instances widget.
They are kept for reference only. Do not conflate them with the current app: the
live widget is the native Swift menu-bar bar in `native/` plus the session hub and
transcript viewer in `lib/`.
